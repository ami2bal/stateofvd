## Propositions (reco programme)

### B4-1 — « Le délai référendaire » (narratif + CE)
- **Pourquoi** : frontière institution / citoyen déjà en préambule S17 ; à **ancrer spatialement** (Chancellerie / FAO) sans urne.
- **Mécanique** : horloge / tag de délai + publication ; pas de nouveau site.
- **Effort** : M · **Valeur formation** : haute (souvent mal compris en App V2).

### B4-2 — « Deux chambres / navette » (si pertinent VD)
- **Pourquoi** : le dépôt parent parle navette ; le plateau GC/CE est déjà là.
- **Mécanique** : allers-retours CE↔GC sur un même objet (EMPL amendé).
- **Effort** : M–L · **Risque** : sur-simplifie le bicaméralisme fédéral — **cadrer en monisme cantonal** (GC seul législatif).

### B4-3 — « Urgence / procédure accélérée »
- **Pourquoi** : contraste avec le cycle normal (tutoriel S1).
- **Mécanique** : moins d’étapes commission, tag « urgence » sur la carte.
- **Effort** : S–M · **Valeur** : contraste pédagogique fort.

### B4-4 — « Classement / non-entrée en matière » (branche déjà partielle S2)
- **Pourquoi** : maximiser les **branches rejectAlt** existantes plutôt qu’un 20ᵉ scénario linéaire.
- **Mécanique** : parcours dédié « le projet meurt en EEM » + fin ✕ déjà UI-ready.
- **Effort** : S · **ROI** : excellent (réutilise S2 + UI Rejouer).

### B4-5 — Mini-parcours « Bureau fixe l’ODJ »
- **Pourquoi** : salle Bureau peu mise en scène comme **décision d’agenda**.
- **Mécanique** : 3–4 étapes GC internes (Bureau → SGC → plénum).
- **Effort** : S · **Valeur** : complète la cartographie spatiale.
